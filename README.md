## Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/uk-grantmakers-dashboard.git
cd uk-grantmakers-dashboard
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
